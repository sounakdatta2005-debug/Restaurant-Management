## 🚀 QUICK START GUIDE

### Step 1: Import Database (2 min)
```bash
mysql -u root -p < schema.sql
```

### Step 2: Configure Database (2 min)
Edit `config/database.php`:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'restaurant_management');
define('DB_USER', 'root');
define('DB_PASS', 'your_password');
```

### Step 3: Setup Apache (2 min)
Set DocumentRoot to:
```
/path/to/restaurant-management/public
```

### Step 4: Access Application (1 min)
```
http://localhost/restaurant-management/public
```

### Step 5: Test Login (1 min)
- Email: admin@restaurant.local
- Password: password

**Total Setup Time**: 5 minutes
